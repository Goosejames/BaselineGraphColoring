Part 1 - Graph Colouring Simulation 
A random geometric graph is generated and each node is assigned a random colour.
Nodes that share the same colour as a neighbour are in conflict and change colour to resolve it.
The simulation tracks how the number of conflicts decreases over time

Part 2 - Colour Count Experiment
We test how the number of avialable colours affects how quickly conflicts are resolved.
The simulation is run multiple times for different colour counts.
The average number of steps to reach a valid colouring is recorded and plotted.
